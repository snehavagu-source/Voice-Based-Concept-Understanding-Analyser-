# Voice-Based Concept Understanding Analyser 🎙️🤖

An innovative web-based application designed to analyze voice inputs and evaluate concept understanding using advanced AI integration.

## Key Features
* **Voice Recording & Upload:** Easy interface to upload audio files for analysis.
* **AI Analysis:** Deep semantic evaluation of user understanding using the Gemini API.
* **Interactive UI:** A clean, user-friendly frontend to visualize analysis results.
* **Database Integration:** Secure storage of analysis records using SQLite.

---

## 🛠️ Tech Stack
* **Backend:** Python, FastAPI
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite
* **AI Integration:** Google Gemini API

---

## 💻 Installation & Setup

Follow these simple steps to run the project locally:

### Step 1: Clone the Repository
```bash
git clone [https://github.com/snehavagu-source/Voice-based-concept-understanding-Analysis.git](https://github.com/snehavagu-source/Voice-based-concept-understanding-Analysis.git)
cd Voice-based-concept-understanding-Analysis

**Step 2: Install Dependencies**
pip install -r Project/requirements.txt

**Step 3: Run the Application**
cd Project
uvicorn main:app --reload
